## Reglas de nombrado

### Presentaciones
> PPT<<X\>>\_U<<X\>>.pdf

### Resumenes
> RESUMEN\_U<<X\>>.<<ext\>> donde ext puede ser (pdf, docx, md)

### Bibliografia
> <<nombre\>>\_<<autor\>>.pdf

### Ejercicios
> <<tema\>>\_caso<<x\>>\_<<alumno\>>.pdf

### Trabajos investigacion
> TI<<XX\>>.pdf

### Trabajos practicos
> TP<<XX\>>.pdf


## Glosario
- PPT: Presentacion
- TI: Trabajo investigacion
- TP: Trabajo practico
- X: numero entero
- tema: tema sobre el que trata el ejercicio
- alumno: quien resolvio el ejercicio
